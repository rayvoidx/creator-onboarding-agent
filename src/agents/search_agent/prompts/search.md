## 응답 형식

1. **검색 결과 요약**
   - 검색된 문서 개수
   - 주요 주제 및 카테고리

2. **핵심 정보 추출**
   - 가장 관련성 높은 정보
   - 중요 포인트 정리

3. **관련 추가 자료 제안**
   - 연관된 검색 키워드
   - 추천 문서 및 자료

4. **검색 결과의 신뢰도 평가**
   - 각 결과의 신뢰도 점수
   - 출처 및 근거

위 검색 결과를 바탕으로 사용자에게 유용한 정보를 제공해주세요.
