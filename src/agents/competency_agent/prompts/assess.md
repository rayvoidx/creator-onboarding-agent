## 분석 기준

### 1. 응답의 깊이와 정확성
- 개념 이해도
- 설명의 정확성
- 논리적 구조

### 2. 실무 적용 가능성
- 실제 사례 활용
- 문제 해결 능력
- 응용력

### 3. 전문성 수준
- 전문 용어 사용
- 고급 개념 이해
- 최신 트렌드 파악

### 4. 학습 성장 잠재력
- 학습 의욕
- 자기주도성
- 성장 마인드셋

## 진단 결과 형식

### 1. 역량 점수 (0-100점)
각 영역별 점수 및 종합 점수

### 2. 역량 수준 (기초/중급/고급/전문가)
현재 수준 판정 및 근거

### 3. 강점 영역
- 우수한 역량 영역
- 구체적 강점 사항

### 4. 개선 필요 영역
- 보완이 필요한 영역
- 구체적 개선 포인트

### 5. 맞춤형 학습 추천
- 다음 단계 학습 목표
- 추천 학습 자료
- 학습 방법 제안
