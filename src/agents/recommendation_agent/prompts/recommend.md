# Learning Recommendation Prompt

## 학습 추천 수행

**사용자 프로필**:
- 역량 수준: {competency_level}
- 학습 선호도: {learning_preferences}
- 관심사: {interests}
- 학습 목표: {learning_goals}

**검색된 관련 자료**: {retrieved_documents}

## 추천 기준

### 1. 사용자 역량 수준에 맞는 난이도
- 현재 수준보다 약간 높은 도전 과제
- 너무 쉽거나 어렵지 않은 적정 난이도
- 점진적 난이도 증가

### 2. 학습 스타일과 선호도
- 선호하는 학습 방식 (영상, 문서, 실습 등)
- 학습 시간 및 속도
- 상호작용 선호도

### 3. 관심사와 목표
- 관심 주제 및 영역
- 단기/장기 학습 목표
- 경력 개발 방향

### 4. 학습 자료의 품질과 평점
- 콘텐츠 신뢰도
- 사용자 평가 및 리뷰
- 최신성 및 정확성

## 추천 결과 형식

### 1. 추천 학습 자료 (3-5개)
각 자료별:
- **제목 및 설명**
- **난이도**: ⭐️⭐️⭐️ (5점 만점)
- **예상 학습 시간**: X시간
- **학습 형식**: 영상/문서/실습/코스

### 2. 각 자료의 추천 이유
왜 이 자료가 적합한지 구체적으로 설명

### 3. 학습 순서 제안
어떤 순서로 학습하면 효과적인지

### 4. 예상 학습 시간
총 학습 시간 및 주차별 계획

### 5. 추가 학습 경로
현재 추천 자료 이후 다음 단계 제안

위 정보를 바탕으로 개인화된 학습 추천을 제공해주세요.
