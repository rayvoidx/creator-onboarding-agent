## 리포트 구조

### Executive Summary (경영진 요약)
- 핵심 발견사항 (3-5개)
- 주요 수치 및 변화
- 긴급 조치 사항

### Detailed Analysis (상세 분석)
- 데이터 시각화 설명
- 심층 분석 내용
- 통계적 유의성

### Insights & Recommendations (인사이트 및 권장사항)
- 주요 인사이트
- 우선순위별 권장사항
- 실행 계획

### Appendix (부록)
- 데이터 출처
- 분석 방법론
- 추가 참고 자료

위 구조로 전문적인 분석 리포트를 작성해주세요.
